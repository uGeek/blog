Gadgetbridge. Emparejando mi Xiaomi Mi Smart Band 4, 5, 6

Lectura: 2 min. 295 palabras. miband

Tengo buenos recuerdos de Gadgetbridge, una aplicación que con mi Xiaomi Mi Smart Band 3, me permitió utilizar la pulsera de un modo libre e independiente, sin tener que verme obligado a utilizar los servidores de XIAOMI.

Cuando apareció la versión 4, 5 y 6, esto cambió. Ahora ya no es tan sencillo ya que necesitamos el token.

Vamos a utilizar un script para obtener el token de acceso Bluetooth de mi Xiaomi Mi Smart Band de los servidores de Huami.

Emparejado Xiaomi Mi Smart Band

Lo primero que haremos será instalar la aplicación de Mi Fit de XIAOMI. Emparejaremos muestra mi band y la actualizaremos.

El segundo paso será instalar la aplicación Zepp, este es un paso importante, para que funcione posteriormente el script.

No te preocupes, estas 2 aplicaciones las desinstalaremos al final.

Abrimos la aplicación Zepp y nos logeamos con la cuenta ver Xiaomi.

Accederemos a perfil-Mis Dispositivos.

Ahora en el apartado Pulsera, si entramos saldrá nuestra pulsera. Nos conectamos con la aplicación Zeep.

Una vez hechos estos pasos, ya estamos listos para utilizar el script.

Instalación Script

Primero clonamos el repositorio


git clone https://github.com/argrento/huami-token.git

accedemos al directorio


cd huami-token

Instalamos todos los paquetes necesarios


sudo pip3 install -r requirements.txt

Para extraer el token, ejecutaremos desde dentro del directorio la siguiente línea:


python3 huami_token.py  --method xiaomi  --email "introduce aqui tu email"  --password "introduce aqui tu contraseña" --bt_keys

Importante

Cada vez que reiniciamos de fabrica la Xiaomi Mi Smart Band, cambia la MAC y Token, al menos en la Xiaomi Mi Smart Band 5. Evita el reiniciarla, sino tendrás que volver instalar las aplicables y ejecutar el script para conseguir el nuevo token.
