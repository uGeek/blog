Servidor SSH en Manjaro

Lectura: 1 min. 147 palabras. ssh

Artículos  Podcasts  Tags

Vamos a ver cómo instalar un servidor SSH en Manjaro.

Instalar OpenSSH


sudo pacman -S openssh

Habilitar e iniciar el servicio SSH

El servicio está instalado siguiendo el paso anterior. Ahora lo que necesitamos es que el servicio corra ininterrumpidamente mediante el daemon. Para ello haremos uso de systemd.


sudo systemctl enable sshd.service

Con este comando habilitaremos el servicio cada vez que iniciemos el sistema operativo, hará que corra el servidor.

Para iniciar el servicio ejecutaremos el siguiente comando.


sudo systemctl start sshd.service

Otras opciones

Con la opción status, podremos ver el estado del servicio.


sudo systemctl status sshd.service

Si necesitas editar el archivo de configuración de SSH, puedes hacerlo del siguiente modo utilizando Nano:


sudo nano /etc/ssh/sshd_config

¿No podemos acceder desde nuestra ip?

Quizas es un problema del cortafuegos. Abre el puerto del servicio con:


sudo ufw allow ssh
