Manjaro. Comandos para terminal y derivadas de Arch Linux

Lectura: 3 min. 420 palabras. arch
Tiempo estimado de lectura: 5 minutos. 596 palabras. arch

Los que venimos de Ubuntu, tenemos problemas al pasar a una dervada de Arch, porque no conocemos bien las instrucciones para actualizar el Sistema Operativo, instalar y demás…

Aquí iré añadiendo periodicamente comandos para ir conociendolos.

Añadir color al gestor pacman

Edita el archivo de configuración:

sudo nano /etc/pacman.conf

Descomenta (quita la #) color.

pacman

Pacman es el sistema de paquetes de Arch oficial, supervisado por ellos. Aquí te muestro un resumen de los comandos más utilizados:

Opción

Descripción

-S

Instala uno o más paquetes.

-R

Desinstala uno o más paquetes.

-Syu

Sincroniza la base de datos de paquetes y actualiza todos los paquetes instalados en el sistema.

-Ss

Busca paquetes en los repositorios disponibles.

-Si

Muestra información detallada sobre un paquete.

-Q

Muestra una lista de paquetes instalados.

-Qi

Muestra información detallada sobre un paquete instalado.

-Qu

Muestra una lista de paquetes que están disponibles para actualizar.

-Sy

Sincroniza la base de datos de paquetes.

-Sw

Descarga un paquete sin instalarlo.

-Syy

Sincroniza la base de datos de paquetes, incluso si ya está sincronizada.

-Sg

Muestra los grupos disponibles y los paquetes que pertenecen a cada grupo.

-Sgg

Muestra una lista de paquetes que pertenecen a un grupo específico.

-Sc

Elimina los paquetes antiguos o huérfanos de la caché de paquetes.

-Scc

Limpia completamente la caché de paquetes.

-Qdt

Muestra una lista de paquetes huérfanos.

yay

yay es un gestor de paquetes AUR (Arch User Repository). A continuación, te muestro una tabla de algunos de los comandos más utilizados con yay:

Comando

Descripción

yay -S <paquete>

Instala un paquete de los repositorios AUR.

yay -Syu

Sincroniza la base de datos de paquetes AUR y actualiza los paquetes instalados en el sistema.

yay -Rs <paquete>

Desinstala un paquete del sistema y sus dependencias no utilizadas.

yay -Ss <palabra>

Busca un paquete en los repositorios AUR.

yay -Si <paquete>

Muestra información detallada sobre un paquete de AUR.

yay -Q

Muestra una lista de paquetes instalados del AUR.

yay -Qi <paquete>

Muestra información detallada sobre un paquete instalado del AUR.

yay -Qua

Muestra una lista de paquetes AUR que están disponibles para actualizar.

yay -Sua

Actualiza todos los paquetes AUR instalados en el sistema.

yay -Pw <paquete>

Descarga un paquete de AUR sin instalarlo.

yay -Yc

Elimina los paquetes huérfanos del sistema.

yay -Ycc

Elimina todos los paquetes de la caché de AUR.

yay -Yua

Actualiza todos los paquetes de la caché de AUR.
