Atajos de teleclado para Firefox y Vimium-FF

Lectura: 5 min. 804 palabras. firefox

Os he hablado muchas veces que no utilizar el ratón en favor del teclado, es una buena práctica ya que nuestra productividad se dispara y permite que en este caso navegar, sea mucho más rápido y fluido.

Vamos a aprender a utilizar los atajos de teclado de Firefox y olvidar para siempre el ratón.

La primera parte del artículo son los atajos de teclado de Firefox, que son totalmente compatibles con el plugin Vimium-FF de la segunda parte del artículo

Atajos de Firefox

Estos son algunos de los atajos de teclado de Firefox

  • Shift + Ctrl + P: Abre ventana de navegación privada.
  • Ctrl + L , F6: Ir a la barra de direcciones
  • Ctrl + K: Ir a la cajetilla de búsqueda, buscar en tu buscador por defecto
  • Alt + Izq: Atrás
  • Alt + Der: Adelante
  • Ctrl + T: Nueva pestaña - Ctrl + W: Cerrar pestaña
  • Ctrl + Shift + W: Cerrar ventana
  • Ctrl + N: Nueva ventana
  • Shift + Enter: Abrir una nueva ventana del navegador
  • Ctrl + Av Pág: Cambia a la pestaña siguiente
  • Ctrl + Re Pág: Cambia a la pestaña anterior
  • Alt + 1-9: Cambia a un número de pestaña
  • Ctrl + Tab, Ctrl + Tab + Izq/Der: Cambiar entre pestañas
  • Ctrl + m: Mutear audio de la pestaña
  • Espacio: Bajar página
  • Shift + Espacio: Subir página
  • Ctrl + F: Buscar
  • Alt + N: Buscar el próximo
  • F5 , Ctrl + r: Refrescar página
  • Ctrl + F5 , Ctrl + Shift + R: Recargar sin caché
  • TAB: Marcar próximo enlace
  • Shift + Tabulador: Marcar enlace anterior
  • Ctrl + arriba: Inicio página
  • Ctrl + abajo: Final página
  • Alt + Inicio: Ir a la página de inicio
  • Shift + Ctrl + T: Vuelve a abrir la última pestaña cerrada (o últimas, si repites).
  • Ctrl + H: Muestra el historial en la barra lateral
  • Ctrl + B: Muestra los marcadores en la barra lateral
  • Ctrl + J: Muestra las descargas en la ventana de catálago
  • Ctrl + D: Añadir página a marcadores
  • Ctrl + Shift + H: Muestra el historial en la ventana de catálogo
  • Ctrl + Shift + B: Muestra los marcadores en la ventana de catálogo
  • Ctrl + Shift + Supr: Borra el historial reciente, Cockies, cache,...
  • Ctrl + +: Aumentar el tamaño del texto
  • Ctrl + -: Disminuir el tamaño del texto
  • Ctrl + 0: Reestablecer el tamaño
  • Ctrl + o: Abrir archivo
  • F11: Pantalla completa
  • F10: Acceder a barra de menú

Vimium-FF

Con esta extensión de Firefox, que también está disponible para derivados de Google Chrome, nos permite utilizar en navegador con los atajos de vim.

Hay una opción que permite deshabilitar esta extensión en las páginas que tu decidas, ya que hay páginas que pueden haber conflictos con los atajos de teclado.

Atajos

Navegación

  • j o Ctrl-e - desplazarse hacia abajo
  • k o Ctrl-y - desplazarse hacia arriba
  • gg - desplazarse hacia arriba
  • G - desplazarse hacia abajo
  • d - desplazarse media página hacia abajo
  • u - desplazarse media página hacia arriba
  • h - desplazarse hacia la izquierda
  • l - desplazarse hacia la derecha
  • r - recargar página
  • yy - copiar URL al portapapeles
  • p - abre la URL en el portapapeles en la pestaña actual
  • P: abre la URL del portapapeles en una nueva pestaña
  • i - entrar en el modo de inserción
  • v - entrar en modo visual
  • gi: enfoca el primer campo de entrada de texto en la página
  • f - abre un enlace en la pestaña actual
  • F - abrir un enlace en una nueva pestaña
  • gf: seleccione el siguiente fotograma de la página
  • gF: seleccione el marco principal / superior de la página

Vomnibarra

  • o - URL abierta, marcador o entrada de historial
  • O - abrir URL, marcador o entrada del historial en una nueva pestaña
  • b - abre un marcador
  • B: abre un marcador en una pestaña nueva
  • T: busca en las pestañas abiertas

Encontrar

  • / - entrar en modo de búsqueda
  • n - avanza hasta la siguiente búsqueda de coincidencias
  • N: retrocede hasta la búsqueda de coincidencias anterior

Navegando por la historia

  • H - retrocede en la historia
  • L - avanza en la historia

Manipular pestañas

  • t - crear una nueva pestaña
  • J o gT: ve una pestaña a la izquierda
  • K o gt: ve una pestaña a la derecha
  • ^ - ir a la pestaña previamente activa
  • g0 - a la primera pestaña
  • g $ - ir a la última pestaña
  • yt - pestaña actual duplicada
  • Alt-P: anclar o desanclar la pestaña actual
  • Alt-m - alternar silencio
  • x - cerrar pestaña actual
  • X - restaurar pestaña cerrada

Puedes encontrar esto atajos en el repositorio de GitHub

Extensión

  • Instala Firefox
  • Extensión para Chrome. Aunque no forma parte de este artículo, si utilizas extensiones de chrome, también puedes instalar esta extensión.
