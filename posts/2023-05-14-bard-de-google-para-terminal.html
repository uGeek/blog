Bard de google para terminal

Lectura: 2 min. 306 palabras. bard

El proyecto de Bard para terminal está liderado por acheong08 en GitHub igual que el de edge-gpt, es una obra de ingeniería inversa de la API de Bard de Google. Permite a los usuarios interactuar con Bard a través de la terminal.

El proyecto Bard está todavía en desarrollo, pero ya ha aprendido a realizar muchos tipos de tareas, como:

  • Seguir instrucciones y completar solicitudes con atención.
  • Responder a preguntas de forma exhaustiva e informativa, incluso si son abiertas, difíciles o extrañas.
  • Generar diferentes formatos creativos de texto, como poemas, código, guiones, piezas musicales, correo electrónico, cartas, etc.

Para usar Bard, necesitarás obtener la Cookie de Bard de Google. Una vez que tengas estos elementos, puedes seguir las instrucciones en la documentación de Bard para empezar.

Recuerda que hay países en los que todavía no funciona Bard. ## Instalación


pip3 install --upgrade GoogleBard

Ayuda


python3 -m Bard -h

Obtener la Coockie

Abre en el navegador la web https://bard.google.com/ - Firefox: Clica el botón derecho del ratón y selecciona la opción Inspeccionar propiedades de accesibilidad > Accede a la sección Storage > Coockies > Copia el número que aparece en el apartado __Secure-1PSID - Otros navegadores: Pulsa la tecla F12 para abrir la consola y copiar de : Ir a la aplicación > Cookies > __Secure-1PSID. Copia el valor de la cookie.

Ejecutar Bard desde la terminal


python3 -m Bard --session "contenido de __Secure-1PSID"

Puedes crear un alias como he hecho yo:


alias bard='python3 -m Bard --session "contenido de __Secure-1PSID"'

Conectándome vía SSH a mi VPS


alias bard='ssh -t usuario@IP_Servidor python3 -m Bard --session "xxxxxxxxxxxxxxxxxxxxxx"'

Utilizando Gotty

Correrá por el puerto 8094, permitirá 3 conexiones simultáneamente y se conectará vía SSH.


gotty -w  -c usuario:contraseña -p 8094  --title-format "Bard"  --max-connection 3  ssh -t vps 'python3 -m Bard --session "xxxxxxxxxxxxxxxxxxxxxxx"
