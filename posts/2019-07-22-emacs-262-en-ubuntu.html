Vamos a instalar Emacs por PPA en Ubuntu 18.04

Instalación por PPA

Añadimos el repositorio:

sudo add-apt-repository ppa:kelleyk/emacs

Actualizamos repositorios:

sudo apt update

Instalamos emacs:

sudo apt install emacs26
