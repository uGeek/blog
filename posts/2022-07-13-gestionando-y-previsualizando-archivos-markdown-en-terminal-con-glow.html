Gestionando y previsualizando archivos markdown en terminal con glow

Lectura: 2 min. 260 palabras. markdown

Artículos  Podcasts  Tags

Glow es un lector para la gestión y previsualización de un modo bonito de archivos markdown. 

Escrito desde cero en Go, Glow destaca por su velocidad en la indexación de notas. Te enamoraras de esta aplicación en segundos.

Sitúate en el directorio donde están tus notas y abre Glow. Ahora disfruta de tus archivos markdown desde la terminal.

Instalación

Instalación en Ubuntu, debian, Raspberry OS,...


echo 'deb [trusted=yes] https://repo.charm.sh/apt/ /' | sudo tee /etc/apt/sources.list.d/charm.list
sudo apt update && sudo apt install glow

Más sistemas operativos en: https://github.com/charmbracelet/glow

también podemos instalarlo en Android con Termux:


pkg install glow

Uso

Previsualización de un archivo


glow README.md


cat README.md | glow -

Previsualización de un directorio

Sitúate en el directorio donde están tus notas y abre Glow.

  • Con la tecla ? podrás ver los atajos.
  • Con / , podrás buscar recursivamente en tus markdown
  • Con q , salir

Leyendo los markdown de repositorios git


# Fetch README from GitHub / GitLab
glow github.com/charmbracelet/glow

# Fetch markdown from HTTP
glow https://host.tld/file.md

Limitar el ancho


glow -w 80

Archivo de configuración

Por defecto el archivo de configuración está en ~/.config/glow/glow.yml. 

No es necesario crearlo, a no ser que quieras tenerlo personalizado.

Este es un ejemplo de archivo de configuración:


# style name or JSON path (default "auto")
style: "light"
# show local files only; no network (TUI-mode only)
local: true
# mouse support (TUI-mode only)
mouse: true
# use pager to display markdown
pager: true
# word-wrap at width
width: 80
