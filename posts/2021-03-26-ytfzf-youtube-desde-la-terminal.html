YTFZF. Youtube desde la terminal

Lectura: 2 min. 260 palabras. bash

La aplicación ytfzf pasa a la primera versión estable, v1.0.0 y ahora permite ver la captura de la imagen de los vídeos, dentro de la terminal.

Este modo utilizar YouTube es muy interesante, ya que permite reproducir vídeos y audios desde YouTube sin consumir demasiado recursos de nuestro PC, ya que utilizaremos para el vídeo un reproductor externo directamente con MPV.

Está previsto que en breve esta aplicación permite a reproducir desde otros sitios web

Instalación

instalar algunas dependencias

Primero instalaremos algunas dependencias:


sudo apt install jq mpv youtube-dl fzf python3-pip


pip3 install ueberzug

Instalación de ytfzf


sudo curl -L "https://raw.githubusercontent.com/pystardust/ytfzf/master/ytfzf" -o /usr/bin/ytfzf && sudo chmod +x /usr/bin/ytfzf

Desde GitHub


git clone https://github.com/pystardust/ytfzf
cd ytfzf
sudo make install

Archivo de configuración

Creamos el directorio de configuración


mkdir -p ~/.config/ytfzf


nano ~/.config/ytfzf/conf.sh

Personalizando el archivo

Voy a utilizar la resolución 1080 y la mayor calidad de audio disponible. Como reproductor, MPV.


YTFZF_HIST=1
YTFZF_LOOP=0
YTFZF_PREF="bestvideo[height<=?1080]+bestaudio/best"
YTFZF_ENABLE_FZF_DEFAULT_OPTS=1
FZF_PLAYER="mpv"

Uso

Buscar con miniaturas

Busca y mira videos con vistas previas en miniatura


ytfzf -t <query>

Mostrar todas las suscripciones con miniaturas (últimas 10)


ytfzf -St

Puedes utilizar varias opciones juntas, aquí hay algunos ejemplos.

Transmite audio (música) y avisa cuando finalice la música


ytfzf -ml <query>

Descarga un video de tu historial


ytfzf -dH

Abrir usando un menú externo en un formato determinado


ytfzf -fD

Si comenzó a ver un video y desea cambiar el formato, primero presione Q para guardar la posición y salir de mpv, luego elija su formato usando


ytfzf -faH

Ayuda


ytfzf -help
