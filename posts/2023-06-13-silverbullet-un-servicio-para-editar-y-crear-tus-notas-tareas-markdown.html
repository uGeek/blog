SilverBullet. Un servicio para editar y crear tus notas, tareas,... Markdown

Lectura: 3 min. 497 palabras. silverbullet

SilverBullet es un servicio muy liviano que permite crear y editar notas en Markdown con muchisimas opciones personalizables.

Es un proyecto de código abierto que nace con la idea de ofrecer una plataforma de escritura basada en Markdown que sea limpia, simple y potente. Su objetivo es facilitar la creación y publicación de documentos en línea, así como la gestión del conocimiento personal mediante el uso de extensiones llamadas /plugs/.

Se basa en el concepto de /espacio/, que es una carpeta donde se almacenan los documentos (notas) en formato Markdown. Los documentos se pueden enlazar entre sí usando la sintaxis [ [enlace a otro documento]], lo que permite crear una red de conocimiento interconectado. Además, se pueden usar etiquetas, comentarios, imágenes y otros elementos para enriquecer el contenido.

SilverBullet se implementa como una PWA (Progressive Web App) que funciona sin conexión después de la primera carga. Esto significa que se puede acceder a los documentos desde cualquier navegador, incluso sin conexión a Internet, y que se sincronizan automáticamente con el servidor cuando hay conexión disponible. También se puede instalar como una aplicación nativa en el escritorio o el móvil.

El nombre de SilverBullet significa bala de plata. Es una metáfora que se usa para referirse a una solución simple y definitiva para un problema complejo o difícil. El origen de esta expresión se remonta a las leyendas sobre los hombres lobo, que supuestamente solo podían ser eliminados con una bala de plata.

¿Qué ventajas tiene SilverBullet respecto otros servicios?

  • Es fácil de usar: solo hay que crear una nueva nota, escribir los documentos en Markdown y publicarlos con un clic.
  • Es extensible: se pueden añadir /plugs/ para ampliar las capacidades de la plataforma, como por ejemplo: generar tablas, gráficos, mapas mentales, diagramas, presentaciones, etc.

Funcionamiento y Atajos

El funcinamiento es muy simple, tal como vas escribiendo en Silver Bullet, si utilizas sintaxis Markdown, al pasa a la siguiente palabra, resaltará la sintaxis.

  • Cuando vuelves hacia atrás, se guarda el contenido de la nota
  • Enlaces: Se pueden crear enlaces entre notas igual que lo hariamos en el método Zetelcasten con [[]]
  • Imagenes: Puedes arrastrar cualquier imagen que veas en una web y se pegará en tu documento. Y lo mejor de todo, se descargará en el directorio raíz donde tenemos todas nuestras notas.
  • Iconos: Es tan simple como escribir : seguido de uno de los atajos al icono.

Hay 2 accesos a menús desde teclado: - Ctrlk: Buscar entre todas tus notas del directorio. Si no existe, escribe el nombre de una pagina para crearla. - Ctrl/: Ejecutar comandos - Alt+h: Ir a página index.md

  • / Tecla de acceso directo para introducir una cabecera (después de escribir el título), tabla, snipet,...
