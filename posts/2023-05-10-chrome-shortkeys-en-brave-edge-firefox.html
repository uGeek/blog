Chrome. Shortkeys en Brave, Edge, Firefox

Lectura: 5 min. 951 palabras. chrome

¿Te gustaría personalizar los atajos de teclado de tu navegador web? ¿Quieres ahorrar tiempo y esfuerzo al navegar por Internet? Entonces te interesa conocer Shortkeys, una extensión gratuita y de código abierto que te permite crear y editar tus propios atajos de teclado para Chrome, Firefox y Edge.

Shortkeys es una herramienta muy útil y fácil de usar que te ofrece muchas posibilidades para mejorar tu experiencia online. Con Shortkeys puedes asignar cualquier combinación de teclas a cualquier acción que quieras realizar en tu navegador, como abrir una nueva pestaña, cerrar la actual, ir a la página anterior o siguiente, recargar la página, buscar una palabra, copiar o pegar texto, activar o desactivar el sonido, cambiar el tamaño del texto y mucho más.

Cómo instalar y configurar Shortkeys

Para empezar a usar Shortkeys, solo tienes que instalar la extensión desde la tienda oficial de tu navegador y acceder a las opciones de configuración. Allí podrás ver una lista de los atajos de teclado predeterminados que ya vienen con la extensión y que puedes modificar o eliminar según tus preferencias. También podrás añadir nuevos atajos de teclado pulsando el botón "Add" y rellenando los campos correspondientes:

  • Keyboard Shortcut: Aquí debes escribir la combinación de teclas que quieres usar para activar el atajo. Puedes usar cualquier letra, número o símbolo del teclado, así como las teclas especiales como Ctrl, Alt, Shift, Tab, Esc, etc. También puedes usar comodines como * o ? para representar cualquier tecla o una sola tecla respectivamente.
  • Behavior: Aquí debes elegir la acción que quieres que se ejecute al pulsar el atajo. Hay muchas opciones disponibles agrupadas en diferentes categorías: Navegación, Pestañas, Ventanas, Marcadores, Historial, Zoom, Sonido, Búsqueda, Selección de texto, Copiar y pegar, JavaScript y Otras. Puedes consultar la descripción de cada opción en la documentación oficial de Shortkeys.
  • Sites: Aquí debes especificar en qué sitios web quieres que el atajo funcione o no funcione. Puedes usar el formato URL completo o parcial, o usar comodines como * o ? para representar cualquier cadena o un solo carácter respectivamente. También puedes usar el símbolo ! al principio para indicar que el atajo no debe funcionar en ese sitio. Por ejemplo: significa que el atajo no funcionará en ninguna página de Google.
  • Active In Inputs: Aquí debes indicar si quieres que el atajo funcione cuando el cursor está dentro de un campo de texto o no. Si marcas esta opción, el atajo se ejecutará incluso si estás escribiendo algo en un formulario o un chat. Si la desmarcas, el atajo se ignorará en esos casos.

Una vez que hayas creado tus atajos de teclado personalizados, solo tienes que guardar los cambios y disfrutar de tu navegación más rápida y cómoda. Recuerda que puedes editar o eliminar tus atajos en cualquier momento desde las opciones de configuración de Shortkeys. También puedes exportar o importar tus atajos en formato JSON para compartirlos con otros usuarios o guardarlos como copia de seguridad.

Cómo usar Shortkeys

Shortkeys es una extensión muy práctica y versátil que te permite adaptar tu navegador web a tus necesidades y gustos. Para usar tus atajos personalizados solo tienes que pulsar las teclas correspondientes mientras estás navegando por Internet. Verás cómo se ejecuta la acción asociada al instante.

Por ejemplo, si has asignado el atajo CtrlT a la acción "Open new tab", cada vez que pulses esas teclas se abrirá una nueva pestaña en tu navegador. Si has asignado el atajo AltS a la acción "Mute/unmute sound", cada vez que pulses esas teclas se activará o desactivará el sonido de la página que estés viendo. Y así con cualquier otro atajo que hayas creado. Puedes consultar la lista completa de tus atajos personalizados desde las opciones de configuración de Shortkeys o desde el icono de la extensión en la barra de herramientas de tu navegador. También puedes desactivar temporalmente todos los atajos pulsando el botón "Disable Shortkeys" en el menú desplegable del icono.

## Accesos directos de teclado compatibles

Los combos clave son cero o más modificadores (ctrl, shift, etc. ) más una tecla normal ( teclas de letras, teclas numéricas, teclas de flecha, etc. ), todas unidas con + signos ( para presionar las teclas al mismo tiempo ) o espacios ( para presionar las teclas una tras otra. Tenga en cuenta que tiene que usar + en lugar de - para indicar teclas de presión al mismo tiempo. Ejemplo: ctrl+b funcionará pero ctrl-b no lo hará. Aquí hay un par de ejemplos:

  • t
  • ctrl+l( mantenga presionado el botón y presione l )
  • t g( presione t y luego g )
  • shift+h
  • ctrl+shift+⇑+pageup
  • alt+f7 r( mantenga presionada la tecla Alt y presione F7, suelte, luego presione r )

Los siguientes modificadores están disponibles: ⇑, shift, option, ⌥, alt, ctrl, control, command, y ⌘. Además de las letras regulares, números y puntuación, se pueden usar las siguientes teclas especiales para accesos directos: backspace, tab, clear, enter, return, esc, escape, space, up, down, left, right, home, end, pageup, pagedown, del, delete y f1 a través de f19.

Conclusión

Shortkeys es una extensión que te permite crear y editar tus propios atajos de teclado para tu navegador web. Con Shortkeys puedes mejorar tu productividad y comodidad al navegar por Internet, ya que puedes asignar cualquier combinación de teclas a cualquier acción que quieras realizar en tu navegador. Shortkeys es gratuita, de código abierto y compatible con Chrome, Firefox y Edge.
