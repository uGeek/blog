Sleek. Una aplicación de todo.txt moderna y elegante

Lectura: 2 min. 368 palabras. todo.txt

Sleek es una aplicación de tareas basada en todo.txt, gratuita y de código abierto. Disponible para Linux, Windows y MacOS.

Los usuarios pueden agregar contextos, proyectos, prioridades, fechas de vencimiento o recurrencias a sus todos y usar estos atributos de todo.txt como filtros o buscarlos mediante una búsqueda de texto completo.

Sleek está comprobando todo el rato el archivo todo.txt para ver si hay cambios, por lo que se puede usar con otras aplicaciones de todo.txt y herramientas de sincronización.

Dispone de modo oscuro y con interfaz en blanco.

La lista de tareas pendientes se puede ordenar y agrupar por prioridades o fechas de vencimiento. Los Todos con fecha de vencimiento o todos repetidos, activarán alarmas con umbrales de 1 o 2 días antes de la fecha de vencimiento. Todas las tareas completas se pueden ocultar o archivar en archivos done.txt.

Instalación

AppImage

Tan sencillo como descargar el último AppImage desde https://github.com/ransome1/sleek/releases.


chmod a+x sleek.AppImage

Ejecutamos con:


./sleek.AppImage

Snap


sudo snap install sleek

Flatpak

Instala sleek desde Flathub usando:


flatpak install flathub com.github.ransome1.sleek

Ejecútalo usando:


flatpak run com.github.ransome1.sleek

Ocultar tareas

Una tarea se puede ocultar (agregar "h: 1") pero sus atributos aparecerán en la lista de filtros y se completarán automáticamente.

Recordatorios

Todos pueden repetirse basándose en una fecha de vencimiento determinada. Puede usar un selector dedicado para agregar la recurrencia o escribirlo a mano:

  • "rec: d" (diario)
  • "rec: w" (semanal)
  • "rec: m" (mensual)
  • "rec: y" (anualmente)
  • también son posibles recurrencias más específicas: "rec: 2d" (cada 2 días)

Las alarmas se activarán cuando vence una tarea mañana o hoy.

Atajos de teclado

Algunos atajos de teclado básicos:

  • Nueva tarea: CTRL + n
  • Buscar todo: CTRL + f
  • Mostrar u ocultar todos los completados: CTRL + h
  • Alternar modo oscuro: CTRL + d
  • Abrir archivo: CTRL + o
  • Abrir configuración: CTRL +,
  • Barra lateral de alternancia: CTRL + b
  • Establecer prioridades (disponible cuando la ventana de agregar / editar está abierta): CTRL MAYÚS [AZ]
  • Enviar todo (disponible cuando la ventana de agregar / editar está abierta): CTRL + Enter
