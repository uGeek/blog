gpt-cli: una herramienta para interactuar con diferentes modelos de lenguaje generativo

Lectura: 2 min. 323 palabras. chatgpt

JH Joselu, nos recomendaba en el grupo de uGeek en Telegram una alternativa a edge-gpt. gpt-cli es una interfaz de línea de comandos chatbot. Esto le permite tener conversaciones interactivas con robots diferentes, incluyendo GPT4 GRATIS.

Instalación

Clona el repositorio


git clone https://github.com/TheLime1/gpt-cli.git

Situate dentro del directorio clonado y:


pip install -r requirements.txt

Ejecutar el GPT CLI app:


python gpt_cli.py

Conseguir - Conectate a https://poe.com/ y logeate a tu cuenta - Pulsa en el candado en la barra de direcciones de brave y selecciona Cookies - Desplega en poe.com y en la carpetita poe pulsa sobre el icono de la galleta - Copia la secuencia de números, letras y símbolos que hay en el apartado Contenido.Pegalos en un archivo que crearemos llamado token.txt dentro del directorio donde está gpt_cli.py.

Ya tienes tus propias Cookies

Iniciando

Ejecuntado gpt-cli


python gpt_cli.py

Opciones

Igual que sucede con edge-gpt, gpt_cli permite exportar las conversaciones que tienes con el chat bot en texto plano


--------------------------------------------------
[1] - Change the bot (somtimes buggy)
[2] - Insert clipboard contents as message
[3] - Export conversation to .txt file
[0] - Close the program

También permite cambiar de bot, insertar texto del porta papeles y cerrar el programa.
