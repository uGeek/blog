Corrector ortográfico en nano

Como cualquier otro editor de texto, nano tiene la posibilidad de habilitar el corrector ortográfico. A diferencia de otros editores, nano no tiene la posibilidad de ir marcando aquellas palabras que no incluye el diccionario del corrector ortográfico y es necesario que una vez acabado el documento, toca hacer la revisión mediante atajos de teclado.

Instalación

La instalación la haremos desde los repositorios.


sudo apt install nano aspell-es

Archivo configuración

El archivo de configuración de nano para el usuario, se encuentra en ~/.nanorc=. Hay un archivo de configuración tipo en la ruta =/etc/nanorc, que podemos copiarlo y partir de este ejemplo:


cp /etc/nanorc ~/.nanorc

Habilitando el corrector ortográfico

Tenemos que descomentar la linea que pone set speller "aspell -x -c"


## Use this spelling checker instead of the internal one.  This option
## does not have a default value.
set speller "aspell -x -c"

Revisión ortográfica

Para hacer la revisión ortográfica, presiona Ctrl + t= y seguidamente =Ctrl + s. Te dará la opción de reemplazar las palabras incorrectas. Selecciona la palabra a reemplazar y pulsa Enter.
