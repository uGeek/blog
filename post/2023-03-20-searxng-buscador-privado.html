SearXNG. Buscador privado

Tiempo estimado de lectura: 9 minutos. 1060 palabras. searxng

¿Qué es SearXNG y por qué deberías usarlo?

SearXNG es un motor de metabúsqueda gratuito que agrega los resultados de más de 70 servicios y bases de datos de búsqueda. Los usuarios no son rastreados ni perfilados. Además, SearXNG se puede usar sobre Tor para el anonimato en línea .

SearXNG es un proyecto impulsado por una comunidad abierta que se inició a mediados del 2021 como una bifurcación del proyecto searx. Su objetivo es proporcionar una alternativa privada y descentralizada a los motores de búsqueda convencionales como Google o Bing.

¿En qué se basa SearXNG?

SearXNG se basa en la idea de que no hay una sola fuente de información que sea la mejor para todas las consultas. Por eso, SearXNG utiliza varios motores de búsqueda y servicios web para obtener los mejores resultados posibles para cada consulta. Algunos ejemplos son Wikipedia, DuckDuckGo, Qwant, YouTube, Reddit, etc.

SearXNG no almacena ningún dato sobre sus usuarios ni sus búsquedas. Tampoco envía ninguna información personal o identificable a los motores de búsqueda o servicios web que utiliza. De esta forma, SearXNG protege la privacidad y evita la personalización o el sesgo de los resultados.

¿Por qué usar SearXNG?

Hay muchas razones para usar SearXNG en lugar de otros motores de búsqueda. Aquí te presentamos algunas:

  • Privacidad: SearXNG respeta tu privacidad y no te rastrea ni te perfila. No guarda ningún registro ni cookie sobre tus búsquedas ni comparte ningún dato con terceros.
  • Diversidad: SearXNG te ofrece una variedad de fuentes y perspectivas para cada consulta. Puedes elegir entre diferentes categorías (general, imágenes, noticias, videos, etc.) y filtrar por idioma, país o fecha.
  • Personalización: SearXNG te permite personalizar tu experiencia de búsqueda según tus preferencias. Puedes cambiar el tema visual, el idioma, el número de resultados por página y los motores de búsqueda que quieres usar.
  • Transparencia: SearXNG es un proyecto abierto y colaborativo que puedes revisar y modificar libremente. El código fuente está disponible en GitHub y la documentación está en docs.searxng.org. También puedes unirte al canal Matrix #searxng:matrix.org para contactar con la comunidad.
  • Independencia: SearXNG es un motor de metabúsqueda descentralizado que no depende de ninguna empresa o entidad centralizada. Cualquiera puede instalar su propia instancia de SearXNG o usar una instancia pública disponible en searx.space.

¿Cómo usar SearXNG?

Usar SearXNG es muy fácil e intuitivo. Solo tienes que acceder a una instancia pública o privada (por ejemplo https://searx.work) e introducir tu consulta en el cuadro de búsqueda.

Puedes usar la sintaxis especial para afinar tu búsqueda con operadores como !, +, -, ", site:, filetype:, etc. Por ejemplo:

  • !wikipedia searxng busca "searxng" en Wikipedia
  • +linux -windows busca páginas que contengan "linux" pero no "windows"
  • "motor de metabúsqueda" busca la frase exacta "motor de metabúsqueda"
  • site:github.com searxng busca páginas del dominio github.com que contengan "searxng"
  • filetype:pdf guía rápida busca archivos PDF que contengan "guía rápida"

También puedes seleccionar la categoría y el idioma que prefieras para tu búsqueda en el menú desplegable situado debajo del cuadro de búsqueda.

Si quieres configurar tus opciones personales (tema visual, idioma predeterminado, motores activados), puedes hacer clic en el icono

Sintaxis de búsqueda

En SearXNG, tienes una sintaxis de búsqueda que te permite modificar las categorías, motores, idiomas y otros aspectos de tu búsqueda. Puedes consultar la lista de motores, categorías e idiomas en las preferencias. Para seleccionar un motor o categoría en particular, utiliza un prefijo de "!" antes del nombre del motor o categoría. Por ejemplo, "!wp" o "!wikipedia" para buscar en Wikipedia, o "!map" para buscar en la categoría de mapas. También puedes utilizar abreviaturas para los motores e idiomas. Los modificadores de motor/categoría son encadenables e inclusivos, lo que significa que puedes combinarlos para obtener resultados más específicos. Para seleccionar el filtro de idioma, utiliza el prefijo ":" antes del código de idioma, como ":fr" para buscar en francés. Además, SearXNG admite el uso de "bangs" externos de DuckDuckGo, lo que te permite buscar directamente en otros sitios web utilizando el prefijo "!!". Sin embargo, ten en cuenta que SearXNG no puede garantizar tu privacidad en estas búsquedas externas.

Ejemplos:

  • Busca en wikipedia paris: !wp paris
  • Buscar en el mapa de categorías para paris: !map paris
  • Búsqueda de imágenes: !images Wau Holland
  • Buscar en la categoría de mapas y DuckDuckGo y Wikipedia para París: !map !ddg !wp paris
  • Buscar en Wikipedia por un idioma personalizado: :fr !wp Wau Holland
  • Buscar en Wikipedia por un idioma personalizado utilizando un "bang" externo: !!wfr Wau Holland

Instancias

Si está buscando instancias en ejecución, listas para usar, visite searx.space . De lo contrario , vaya a los manuales de usuario , administrador y desarrollador que encontrará en nuestra página de inicio .

Docker Compose

  • Arranca el docker, añadiendo almohadilla ante los volumentes: Esta instrucción sirve para iniciar el contenedor de docker con la imagen de searxng, y montar los volúmenes especificados en el archivo docker-compose.yml. La almohadilla indica que se trata de un comentario y no se ejecuta como comando.
  • Copiar la configuración una vez arrancado el docker: Esta instrucción sirve para copiar el archivo de configuración de searxng desde el contenedor al directorio local. Esto permite modificar los parámetros de searxng según las preferencias del usuario.
  • docker cp searxng:/etc/searxng searxng/: Esta es la sintaxis del comando docker cp, que copia archivos o directorios entre un contenedor y el sistema de archivos local. En este caso, se copia el directorio /etc/searxng desde el contenedor llamado searxng al directorio local con el mismo nombre.



searxng:
container_name: searxng
image: searxng/searxng:latest
ports:
- "8080:8080"
volumes:
- ./searxng:/etc/searxng:rw
environment:
- SEARXNG_BASE_URL=http://192.168.1.100:8080
- INSTANCE_NAME=SearXNG

Ejemplo de más configuración

Exporta las búsquedas en diferentes formatos. Los formatos disponibles son html, rss, csv y json, permitiendo guardar las búsquedas como páginas web, feeds de noticias, tablas o objetos de texto.

search:
# Filter results. 0: None, 1: Moderate, 2: Strict
safe_search: 2
# Existing autocomplete backends: "dbpedia", "duckduckgo", "google", "yandex",
# "seznam", "startpage", "swisscows", "qwant", "wikipedia" - leave blank to turn it off
# by default.
autocomplete: "duckduckgo"
# minimun characters to type before autocompleter starts
autocomplete_min: 3
# Default search language - leave blank to detect from browser information or
# use codes from 'languages.py'
default_lang: "es"
# Available languages
# languages:
#   - all
#   - en
#   - en-US
#   - de
#   - it-IT
#   - fr
#   - fr-BE
# ban time in seconds after engine errors
ban_time_on_fail: 5
# max ban time in seconds after engine errors
max_ban_time_on_fail: 120
# remove format to deny access, use lower case.
# formats: [html, csv, json, rss]
formats:
- html
- rss
- csv
- json
