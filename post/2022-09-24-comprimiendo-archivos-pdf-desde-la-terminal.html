Comprimiendo archivos pdf desde la terminal

Tiempo estimado de lectura: 2 minutos. 321 palabras. terminal

Es probable que más de una vez te hayas visto con la necesidad de enviar un archivo pdf por internet y el formulario te limita el tamaño, o simplemente, quieras reducir el tamaño de archivos de tu biblioteca digital.

Las aplicaciones de scanear en el móvil, no suele venir con la opción para comprimir pdf, pero esto no va a ser un problema para nosotros.

Gracias al paquete ghostscript, vamos a comprimir archivos pdf desde la terminal.

Es muy probable que este paquete ya esté instalado en tu máquina, pero si no es así, lo instalaremos de un modo sencillo desde los propios repositorios.

Instalación

sudo apt install ghostscript

Comado de ejemplo

Este comando va a comprimir muchísimo y de un modo legible, el archivo pdf.

entrada.pdf, es el archivo origen que queremos comprimir y lo convertirá en el archivo salida.pdf.

gs -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 -dPDFSETTINGS=/screen -dNOPAUSE -dQUIET -dBATCH -sOutputFile=salida.pdf entrada.pdf

Opciones

  • -dPDFSETTINGS Opción Descripción
  • -dPDFSETTINGS=/screen Tiene una calidad inferior y un tamaño más pequeño. ( 72 ppp )
  • -dPDFSETTINGS=/ebook Tiene mejor calidad, pero tiene un tamaño un poco más grande ( 150 dpi )
  • -dPDFSETTINGS=/prepress La salida es de mayor tamaño y calidad ( 300 ppp )
  • -dPDFSETTINGS=/printer La salida es de una calidad de tipo de impresora ( 300 ppp )
  • -dPDFSETTINGS=/default Selecciona la salida que es útil para múltiples propósitos. Puede causar PDFS grandes.
