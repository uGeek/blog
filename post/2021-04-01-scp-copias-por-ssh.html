SCP. Copias por SSH

scp, comando que signigica secure copy, permiten copiar un archivo o directorio desde una máquina origen a una máquina destino y viceversa, utilizando el servidor SSH. scp origen destino

La estructura del comando sería algo así como:


scp usuario@ip:ruta_del_archivo destino

Este sería un ejemplo:


scp archivo.txt angel@192.168.1.100:/home/angel/Documentos/

Ejemplo:


scp angel@192.168.1.100:/home/angel/archivo.txt /home/angel/Documentos

Copiaremos el archivo.txt del servidor dentro del directorio Documentos.

A diferencia de ssh, scp usa el conmutador P mayúscula para configurar el puerto en lugar de la p minúscula. En este ejemplo utilizaremos el puerto 100


scp -P 100

Para copiar un directorio, utilizaremos -r de modo recursiva


scp -r

Ejemplo:


scp -r -P 2222 angel@192.168.1.100:/home/angel/Documentos/varios /home/angel/Documentos/

El parámetro -l permite limitar la velocidad en Kbps. Ejemplo: 8000Kbps


scp -r -l "8000" -P 2222 angel@192.168.1.100:/home/angel/Documentos/varios /home/angel/Documentos/
