unrar en Raspberry Pi

Tiempo estimado de lectura: 1 minutos. 182 palabras. bash

No todo el software está disponible para ARM, como por ejemplo unrar. Vamos a ver como poder descomprimir archivos desde nuestra Raspberry.

./images-blog/unrar

Instalación

Descargaremos unrar desde la web https://www.rarlab.com/rar_add.htm

En este caso encontramos en el largo listado, UnRAR for ARM Linux Command line freeware ARM Linux UnRAR.

Vamos a descargarlo:

wget https://www.rarlab.com/rar/unrar-5.5.0-arm.gz

Descomprimimos:

gunzip unrar-5.5.0-arm.gz

Damos permisos de ejecución:

chmod +x  unrar-5.5.0-arm

Ahora podemos ejecutarlo con:

./unrar-5.5.0-arm

Descomprimir

Para descomprimir, utilizaremos la opción x.

./unrar-5.5.0-arm x archivo.rar
