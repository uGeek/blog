SSH. Eliminar la clave del host en un servidor remoto

Tiempo estimado de lectura: 1 minutos. 205 palabras. ssh

Cuando se ha cambiado la dirección IP del servidor o cuando se ha reinstalado el sistema operativo en el servidor, es necesario eliminar la clave del host, para obligar al cliente SSH a volver a comprobar la identidad del servidor la próxima vez que se conecte.

Con el comando ssh-keygen -R IP_Servidor eliminaremos la clave del host de un servidor remoto.

Por ejemplo, si deseas eliminar la clave del host para un servidor con dirección IP 192.168.1.100, es tan sencillo como ejecutar el siguiente comando:

ssh-keygen -R 192.168.1.100

Ahora ya podrás conectarte sin problemas….
