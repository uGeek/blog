Docker: Raneto

Tiempo estimado de lectura: 1 minutos. 164 palabras. docker

Raneto es una plataforma de base de conocimientos de código abierto que utiliza archivos Markdown estáticos para potenciar su base de conocimientos.

Docker Compose

El docker-compose para construir reneto:

version: "2.1"
services:
raneto:
image: linuxserver/raneto
container_name: raneto
environment:
- PUID=1000
- PGID=1000
- TZ=Europe/Madrid
volumes:
- $HOME/reneto/appdata:/config
ports:
- 3000:3000
restart: unless-stopped
