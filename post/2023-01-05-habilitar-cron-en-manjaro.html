Habilitar Cron en Manjaro

Tiempo estimado de lectura: 1 minutos. 209 palabras. cron

Después de instalar Manjaro con el escritorio gnome, he abierto mi archivo cron para editar mis scripts automatizados y… He aquí mi sorpresa!, el servicio estaba detenido.

Para que todo funcionara, he tenido que habilitar cron, pero previamente he comprobado si estaba instalado.

Aquí te explico lo que he hecho.

Comprobar si el paquete estaba instalado

Verifica si ya está instalado mediante el siguiente comando:

pacman -Qs cron

o

pacman -Qs cronie

Instalación

Si no está instalado…

sudo pacman -S cronie

Servicio cron

Ahora comprobaremos si cron se está ejecutando:

systemctl status cronie

Para habilitar el servicio, lo haremos así:

sudo systemctl enable cronie
