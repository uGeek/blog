Cómo hacer funcionar la interfaz de audio USB Scarlett 2i2 3.ª gen. de Focusrite en Linux

Tiempo estimado de lectura: 3 minutos. 371 palabras. scarlett

La interfaz de audio USB Scarlett 2i2 3.ª gen. de Focusrite es una interfaz de audio de 2 entradas y 2 salidas con conexión USB-C que permite grabar y reproducir audio con alta calidad y bajo ruido . En este artículo te explicare cómo hacerla funcionar en Linux.

Inhabilitar el montado como interfaz de almacenamiento

Cuando conectamos por primera vez nuestra interfaz Scarlett 2i2, por defecto se monta en nuestro sistema linux como si fuera una unidad de almacenamiento USB. Dentro de este directorio hay una serie de archivos para registrar nuestra interfaz. Para que esto no suceda y funcione correctamente en el sistema linux, es necesario seguir los siguientes pasos: - Desconecta la interfaz del cable usb - Mantén pulsado el botón de 48v durante un mínimo de 5 segundo a la vez que conectes el cable USB-C al PC. - Ahora ya no volverá a montarse al inicio como una unidad de almacenamiento y si como interfaz de sonido

Configurar la interfaz Scarlett en tu sistema linux

  • Localiza tu modelo y copia el texto que hay a continuación para añadirlo a un archivo de configuración que vamos a crear.

Gen 2:

  • 6i6: options snd_usb_audio vid=0x1235 pid=0x8203 device_setup=1
  • 18i8: options snd_usb_audio vid=0x1235 pid=0x8204 device_setup=1
  • 18i20: options snd_usb_audio vid=0x1235 pid=0x8201 device_setup=1

Gen 3:

  • Solo: options snd_usb_audio vid=0x1235 pid=0x8211 device_setup=1
  • 2i2: options snd_usb_audio vid=0x1235 pid=0x8210 device_setup=1
  • 4i4: options snd_usb_audio vid=0x1235 pid=0x8212 device_setup=1
  • 8i6: options snd_usb_audio vid=0x1235 pid=0x8213 device_setup=1
  • 18i8: options snd_usb_audio vid=0x1235 pid=0x8214 device_setup=1
  • 18i20: options snd_usb_audio vid=0x1235 pid=0x8215 device_setup=1

Vamos a crear una archivo de configuración, es muy importante que acabe con la extensión .conf.

sudo nano /etc/modprobe.d/scarlett.conf

Añade la línea de la interfaz que tengas, en mi caso la 2i2 3ª Generación

En mi caso pego:

options snd_usb_audio vid=0x1235 pid=0x8210 device_setup=1

Reinicia el PC

Alsamixer

Vamos a ver que todo funciona correctamente. Para ello abriremos nuevamente nuestra terminal y ejecutaremos alsamixer: - Pulsaremos F6 para seleccionar la tarjeta a utilizar (en mi caso Scarlett 2i2 USB) - Ahora pulsa F5 para ver todos las opciones
