Ampliar memoria SWAP

Tiempo estimado de lectura: 2 minutos. 325 palabras. raspberry

Tener poca memoria swap, puede hacer que se detengan determinados servicios o comience a ir lento nuestro servidor o PC.

Vamos a ver como ampliar la memoria swap de los 100Mb que trae Raspbian OS por defecto en su versión Lite a 512Mb.

Conocer cuante SWAP tenemos

Para comprobar la memoria swap que tenemos y ver su estado, es tan sencillo como utilizar htop.

También podemos utilizar el siguiente comando:

cat /proc/meminfo | grep Swap

Este comando nos mostrará tres valores:

SwapCached:  # swap ocupada
SwapTotal:   # swap disponible
SwapFree:    # swap libre

Aumentar SWAP

Para aumenta la swap, editaremos el siguiente archivo:

sudo nano /etc/dphys-swapfile

En el caso de tener solo 100Mb, como es mi caso, veremos la siguiente variable con este valor:

CONF_SWAPSIZE=100

Ahora cambiaremos el valor por 512Mb, que es nuestro objetivo:

CONF_SWAPSIZE=512

Implementar la nueva SWAP

Para implementar esta nueva swap tenemos dos modos de hacerlo.

  • La primera, que es la que más me gusta, es reiniciar nuestro servidor.

  • La segunda, nos permite mediante dos comando por una parte eliminar la swap y después implementar el nuevo valor. Es un modo útil si no puedes detener el servidor y no puedes reiniciar.

sudo dphys-swapfile setup
sudo dphys-swapfile swapon

Conclusión

Si tienes una Raspberry 4 con 4 o 8 Gb, no es necesario ampliar la swap, pero si es cierto que para la Raspberry 3 o si tienes muchos servicios corriendo, nunca está de más.
