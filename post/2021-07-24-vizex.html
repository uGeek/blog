Vizex. Ver almacenamiento disponible y ocupado de tus discos

Tiempo estimado de lectura: 1 minutos. 191 palabras. discoduro

Soy amante de las aplicaciones que te muestran todo el espacio que tienes en tu disco duro. Ya he escrito en varios artículos diferentes aplicaciones, hoy toca el turno a vizex.

sudo pip3 install vizex

En ubuntu 20.04 me ha funcionado sin problemas, en cambio, en Raspberry OS me daba el siguiente error Original error was: libf77blas.so.3: cannot open shared object file: No such file or directory , así que he tenido que instalar:

sudo apt-get install libatlas-base-dev

Para actualizar la aplicación, lo haremos mediante:

sudo pip3 install vizex --upgrade

Uso

Con el comando vizex, nos muestra el almacenamiento ocupado y libre de todas las unidades que tenemos

vizex

Si ejecutamos vizexdf, nos mostrará todos los archivos o directorios de la ubicación actual, con el tamaño

vizexdf

-a, muestra los archivos y directorios ocultos -d, ordenar columnas en orden descendente -s, ordena la tabla por tipo, tamaño, nombre o fecha -tree, imprime un árbol

Ayuda

Como siempre, podemos acudir al comando de ayuda para ver todas las opciones:

vizex --help
