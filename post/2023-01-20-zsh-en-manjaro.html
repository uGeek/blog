ZSH en Manjaro

Voy a instalar ZSH en la terminal de mi Raspberry Pi con Manjaro

Instalación


sudo pacman -S zsh git gedit

Instalación de Oh My Zsh


sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"


sudo chsh -s /bin/zsh usuario

Ejemplo: Cambiar a ZSH al usuario angel


sudo chsh -s /bin/zsh angel

Cambia el tema

Si deseas cambiar el tema de ZSH, Edita:


nano ~/.zshrc

Para usar un tema diferente, cambie el valor para que coincida con el nombre del tema deseado.\

Por ejemplo:


ZSH_THEME="agnoster" # (this is one of the fancy ones)
