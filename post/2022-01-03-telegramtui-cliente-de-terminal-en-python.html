TelegramTUI. Cliente de Terminal en Python

Tiempo estimado de lectura: 4 minutos. 514 palabras. telegram

Telegram-cli es una versión de Telegram que quizás te sea compleja de utilizar en terminal. Telegramtui es otro cliente para terminal que es más parecido a otros clientes de mensajería para terminal, dividido principalmente en 3 partes. A la Izquierda los chats, a la derecha superior la conversación y justo en la parte inferior, la cajetilla donde introduciremos el texto.

Esta versión puedes utilizarla tanto en tu PC como Raspberry Pi ya que es compatible con arquitectura amd64 así como ARM.

Vamos a ver como instalarla

Instalación

sudo pip3 install telegramtui

Telegramtui tiene las siguientes dependencias, que serán instaladas con el comando anterior:

  • Teletón
  • NpyScreen
  • python-aalib

Archivo de Configuración

Ahora tendremos que ir al archivo de configuración y editarlo.

nano ~/.config/telegramtui/telegramtui.ini

Y encontraremos esto:

[telegram_api]
; obtain api keys here: https://core.telegram.org/api/obtaining_api_id
; only api_id and api_hash are needed for the application to work
api_id = *api_id here*
api_hash = *api_hash here*
; number of thread to receive data from tg api
workers = 3
; you can use few accounts, change session name
session_name = user

[app]
name = TelegramTUI v0.1.2
message_dialog_len = 50

[proxy]
type = *[SOCKS4|SOCKS5|HTTP]*
addr = *ip add here*
port = *port here*
; set username and password (only for socks)
username = None
password = None

[other]
timezone = +0

Tenemos que rellenar el api_id = *api_id here* y api_hash = *api_hash here*, que podremos conseguirlo desde la siguiente página de Telegram.

Generar claves API de Telegram: 1. Iremos a la url https://my.telegram.org iniciamos sesión con nuestro número de teléfono, nos enviará un mensaje a otro cliente de Telegram con un código. 2. Vamos a API Development tools. 3. Nos dará la opción de crear una nueva aplicación. Ponemos el título y nombre. 4. Le daremos a crear y guardaremos api_id y api_hash.

Este api_id y api_hash lo pondremos en el archivo de configuración. Recuerda que hemos creado una aplicación estos datos pueden utilizarse con diferentes aplicaciones y números de telefono, pero no lo compartas.

### Atajos

  • Navegación: Tab, Shift+Tab, Mouse
  • Enviar mensaje: Ctrl+S, Alt+Enter
  • Borrar mensaje: Ctrl+R
  • Enviar archivo: Ctrl+O
  • Salida: Ctrl+Q, ESC
  • Dupdo: Shift+Mouse
  • Pegar: Shift+Ins, Shift+Middle mouse button
