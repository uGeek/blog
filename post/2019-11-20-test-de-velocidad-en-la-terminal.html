Test de velocidad en la terminal

Tiempo estimado de lectura: 1 minutos. 178 palabras. terminal

Comprobar la velocidad de tu conexión a internet desde tu terminal es muy sencillo. Como puedes ver, para hacerlo, no necesitas necesariamente utilizar el navegador web.

Vamos a utilizar speedtest-cli, una librería en python que de un modo muy sencillo, nos va a proporcionar esta información

Instalando pip

sudo apt install python3-pip

Instalando speedtest-cli

pip3 install speedtest-cli

Comprobando la velocidad

Tan sencillo como:

speedtest-cli

Ayuda

Encontrarás muchas mas opciones, en la ayuda.

speedtest-cli -h
