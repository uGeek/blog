Configuración y diferencias entre bashrc, bash profile, bashrc, profile o sus versiones zsh

Tiempo estimado de lectura: 2 minutos. 273 palabras. bash

Navegando por las web encontré este interesante Post donde se explica la diferencia entre los archivos de configuración ~/.bashrc, ~/bash_profile, /etc/bashrc, /etc/profile.

La diferencia más importante, es que unos archivos afectan solamente al usuario que está ejecutando la sesión y los otros, afectan a la totalidad de usuarios.

En estos archivos, podemos añadir scripts para que se ejecuten al iniciar el usuario o sesión.

Vamos a ver como configurarlo y como afecta cada archivo. Lo dividiremos en los dos grandes grupos.

Todos los usuarios

Cuando inicia la sesión

Archivo /etc/profile

sudo nano /etc/profile

Cuando se ejecuta Bash o Zsh

Archivo /etc/bashrc

En Bash

sudo nano /etc/zsh/zshrc

En Zsh

sudo nano /etc/zsh/zshrc

Para tu usuario

Cuando inicias sesión

En Bash

Archivo ~/.bash_profile

nano ~/.bash_profile

En Zsh

Archivo ~/.zsh_profile

nano ~/.zsh_profile

Cuando inicias Bash o Zsh

En Bash

Archivo ~/.bashrc

nano  ~/.bashrc

En Zsh

Archivo ~/.zshrc

nano  ~/.zshrc
