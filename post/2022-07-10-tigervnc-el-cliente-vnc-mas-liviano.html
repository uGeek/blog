TigerVNC. El cliente vnc más liviano

En ese testeo de probar diferentes clientes para vnc, hoy toca el turno de TigerVNC.

Lo más interesante de este cliente, es que el cliente funciona a pantalla completa mostrando el escritorio remoto, maximizando la pantalla sin ningún tipo de pestaña, teniendo la sensación de trabajar directamente en el PC remoto  saliendo solo  pulsando la tecla F8.

Instalación

Podemos encontrar TigerVNC en los repositorios de ubunutu


sudo apt-get install -y tigervnc-viewer

Para introducir la contraseña en formato texto con un único comando, es necesario instalar el paquete tigervnc-common


sudo apt install tigervnc-common

Manjaro

En manjaro lo instalaremos con:


sudo pacman -S tigervnc

Abrir TigerVNC

Tan sencillo como lanzar el comando desde nuestro escritorio.

Si queremos hacerlo desde la terminal, podemos entrar directamente a nuestro servidor vnc del siguiente modo:


vncviewer 192.168.1.102:5901 -passwd <(echo "contraseña" | vncpasswd -f)

En este ejemplo, el servidor está corriendo por el puerto 5901= y la ip del servidor es =192.168.1.102

Haciendo de este modo, creando por ejemplo un alias, entraremos directetamente a nuestro escritorio remoto con VNC

Si deseas lanzar en pantalla completa, es necesario utilizar la opción -FullScreenMode


vncviewer 192.168.1.102:5901 -passwd <(echo "contraseña" | vncpasswd -f) -FullScreen

Uso

F8 es la tecla que te permitirá acceder al menú de TigerVNC. F8 también es la tecla para salir del escritorio remoto.

Atajos:

  • F8+ f = Pantalla completa

El cursor desaparece

Es muy muy importante, sobretodo en PC's con Windows, Dejar conectado un ratón en la torre. Si no está conectado, depende del PC, pero es posible que no funciones el teclado númerico del teclado y no se vea el cursor del ratón.

Si desaparece el cursor dependiendo del escritorio que utilices, puedes habilitar el cursor de TigerVNC que es un punto. Para ello, pulsando F8, seleccionaremos la pestaña Misc. de Miscelánea y marcaremos la opción Show dot when no cursor. 

Ejemplo de otro comando para lanzar tigervnc

Me pareción interesante dejar esta opción para lanzar tigervnc de un usuario. 


vncviewer -viewonly -DotWhenNoCursor=1 -QualityLevel=9 -NoJPEG -PreferredEncoding=Raw -CompressLevel=0 -geometry 1920x1080+0+0 -AutoSelect=0 -Fullscreen=1 -FullscreenSystemKeys=0 [HOSTNAME]::2222[0- 1]
