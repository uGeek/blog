Joplin para terminal, llega a snap

Tiempo estimado de lectura: 2 minutos. 249 palabras. joplin

Si te gusta tener Joplin en tu terminal y no te gusta complicarte la vida instalando nodejs, etc… Ahora puedes hacerlo desde snap de un modo sencillo. Este paquete trae los editores de texto nvim y nano. Una de las desventajas, es no poder utilizar el editor de texto que desees.

Yo personalmente, lo he instalado en la terminal tal como explicaba en el artículo Joplin Desktop, Joplin para Terminal y WebClipper y últimamente utilizo de editor de texto typora.

Aún así, vamos a ver como instalarlo y utilizarlo

Instalación

Tan sencillo como:

sudo snap install joplin

Ahora escribiendo joplin en la terminal, ya tendremos el cliente corriendo.

Editores

El editor predeterminado se puede cambiar usando:

joplin config editor /snap/joplin/current/usr/bin/nvim # default
joplin config editor /snap/joplin/current/bin/nano # easier alternative

Editores fuera de snap

También podemos utilizar editores de texto plano fuera de los paquetes snap.

En este ejemplo utilizaré Emacs con un archivo de configuración personalizado para la versión de terminal.

joplin config editor "/usr/bin/emacs -nw  -q -l ~/.emacs.d/init-terminal.el"

Habilitar servidor de joplin

El servidor no se está ejecutando de forma predeterminada, pero se puede habilitar con:

snap set joplin server.enabled=true
