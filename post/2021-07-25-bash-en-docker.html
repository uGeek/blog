Bash en Docker

Bash viene en cualquier distro linux. De todas maneras, podemos ejecutarlo con Docker. Vamos a ver como ejecutar desde el docker el script script.sh.

Corriendo bash desde un docker

Vamos a correr la versión 5.0=. Mira [[https://hub.docker.com/_/bash?tab=tags&page=1&orderinglast_updated][las etiquetas]] para personalizar la versión.


docker run -it --rm -v $HOME/docker/bash/script.sh:/script.sh:ro bash:5.0 bash /script.sh

También podemos utilizar la última versión:


docker run -it --rm -v /root:/script.sh:ro bash bash
