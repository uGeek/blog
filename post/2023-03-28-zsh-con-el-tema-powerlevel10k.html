ZSH con el tema PowerLevel10k

Tiempo estimado de lectura: 2 minutos. 329 palabras. zsh

Tema alucinante de ZSH con PowerLevel10k

PowerLevel10k es un tema para el intérprete de comandos en sistemas operativos Unix y Linux que permite personalizar la apariencia de Zsh. PowerLevel10k se enfoca en la velocidad, la flexibilidad y la experiencia de usuario permitiendo cambiar el color, el estilo y los iconos de la terminal. Además, PowerLevel10k tiene un asistente que ayuda a configurar el tema según las preferencias del usuario.

Clona el repositorio:

git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

Iniciando PowerLevel10k

La primera vez que ejecutemos PowerLevel10k, nos aparecerá un asistente para personalizar el tema y dejarlo a nuestro gusto.

Este comando nos lanzará el asistente y comenzará con configuración:

p10k configure

Si no te funciona, también puedes ejecutar el siguiente comando con:

source ~/.oh-my-zsh/custom/themes/powerlevel10k/powerlevel10k.zsh-theme

Añade el tema PowerLevel10k

Añade el nuevo tema en el archivo ~/.zshrc con:

ZSH_THEME="powerlevel10k/powerlevel10k"

Si no te funciona, siempre podrías añadir al final del archivo la siguiente línea ~/.zshrc.

source ~/.oh-my-zsh/custom/themes/powerlevel10k/powerlevel10k.zsh-theme

Modificar el tema PowerLevel10k

Si no te gusta como ha quedado tu configuración, modifica con el comando:

p10k configure

Instalación de fuentes y símbolos Arch, Manjaro

Si tienes problemas con las fuentes y símbolos, que no se previsualizan, quizás necesitas instalar estos siguientes paquetes

sudo pacman -S powerline-common awesome-terminal-fonts
yay -S --noconfirm ttf-meslo-nerd-font-powerlevel10k

Fuente: https://github.com/romkatv/powerlevel10k

Instalación de fuentes y símbolos en termux (Android)

mkdir -p ~/.termux

curl -fsSL -o ~/.termux/font.ttf 'https://github.com/uGeek/zsh/blob/main/fonts/NerdFonts/MesloLGS%20NF%20Regular.ttf?raw=true'

termux-reload-settings

Para volver al estado inicial:

rm -f ~/.termux/font.ttf && termux-reload-settings
