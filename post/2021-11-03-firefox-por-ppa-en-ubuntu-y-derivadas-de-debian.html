Firefox por PPA en Ubuntu y derivadas de Debian

Tiempo estimado de lectura: 1 minutos. 137 palabras. firefox

Con la nueva versión 94 de Firefox, ya no necesitamos de plugins para gestionar la memoria que ocupa tener pestañas abiertas. 

Firefox, pasado un tiempo determinado sin uso de una pestaña, liberará la memoria para gestionar mejor así la memoria de tu PC.

Si quieres gestionar manualmente, porque así lo deseas, tienes que acceder escribiendo en la barra donde introducimos las urls, about:unloads

Instalación

Pero para probar esta nueva característica, pasa por instalar la última versión. 

Para instalar Firefox via PPA Oficial en Ubuntu, Debian y derivadas, escribiremos en la terminal:

sudo add-apt-repository ppa:ubuntu-mozilla-security/ppa
sudo apt-get update
sudo apt-get upgrade
