Vinagre. Cliente VNC para Linux

Un cliente VNC creado por el proyecto GNOME que te permite conectarte a un escritorio que se ha compartido en otro PC. Vinagre es uno de los clientes que venía antiguamente por defecto en Ubuntu.

Conexiones

  1. VNC: Virtual Network Computing
  2. RDP: protocolo de escritorio remoto. Remote Desktop Protocol
  3. SPICE: Simple Protocol for Independent Computing Environments
  4. SSH: Secure Shell

 

Características

  • Se pueden realizar múltiples conexiones, cada una tiene su propia pestaña.
  • Las contraseñas para las conexiones se pueden recordar.
  • Las conexiones se pueden marcar.
  • La pantalla completa se activa o desactiva presionando F11.
  • Puede guardar capturas de pantalla de la conexión actual.

 

 

Instalación

 


sudo apt install vinagre

 

Uso

Desde la terminal podemos arrancar vinagre en pantalla completa con:


vinagre -f 192.168.1.102:5901

Si no quieres lanzarlo desde la terminal, búscalo en el menú de tu distro como Visor de escritorio remoto en un lanzador de aplicaciones escribe vinagre.
