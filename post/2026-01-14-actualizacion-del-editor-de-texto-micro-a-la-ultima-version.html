Actualización del Editor de Texto Micro a la Última Versión

Tiempo estimado de lectura: 0 minutos. 0 palabras. micro

Este artículo explica detalladamente el procedimiento necesario para actualizar el editor de texto Micro, asegurándose de que la instalación incluya el binario más reciente obtenido directamente desde el repositorio oficial.

Descarga e Instalación del Binario

Para asegurar que el editor de texto Micro esté a la última versión, es indispensable obtener el archivo binario compatible con la arquitectura del sistema directamente desde el repositorio oficial de GitHub.

Paso 1: Descargar el archivo binario

Se recomienda utilizar el comando wget para obtener la versión más reciente. A continuación, se muestra un ejemplo utilizando la versión 2.0.15 para la arquitectura linux-arm64:

wget github.com/zyedidia/micro/releases/download/v2.0.15/micro-2.0.15-linux-arm64.tar.gz

Paso 2: Descomprimir el archivo

Una vez descargado, se debe utilizar el comando tar para extraer el contenido del archivo comprimido:

tar -xzf micro-2.0.15-linux-arm64.tar.gz

Paso 3: Ejecución de prueba

Se aconseja acceder al nuevo directorio creado y ejecutar el binario directamente para confirmar que la instalación se realizó con éxito:

./micro

Configuración: Añadir Micro al PATH (Shell Fish)

Para permitir la ejecución del editor micro desde cualquier ubicación de la terminal (sin necesidad de especificar la ruta completa), es necesario añadir el directorio que contiene el binario a la variable de entorno PATH dentro de la configuración de Fish Shell.

Paso 1: Editar el archivo de configuración de Fish

El usuario debe acceder y modificar el archivo de configuración de Fish Shell, generalmente ubicado en el directorio de usuario:

nano ~/.config/fish/config.fish

Paso 2: Añadir la ruta del binario

Se debe insertar la siguiente línea dentro del archivo de configuración. Es fundamental verificar que la ruta /nube/scripts/local/micro/micro-2.0.15/ coincida exactamente con la ubicación donde se extrajo el editor en el paso anterior:

fish_add_path /nube/scripts/local/micro/micro-2.0.15/

Paso 3: Recargar la configuración

Después de guardar los cambios en el archivo config.fish, se debe recargar la configuración de Fish Shell para que las modificaciones en el PATH surtan efecto inmediatamente:

source ~/.config/fish/config.fish

Configuración para zsh

nano ~/.zshrc

y agrega:

export PATH="$PATH:/nube/scripts/local/micro/micro-2.0.15/"
