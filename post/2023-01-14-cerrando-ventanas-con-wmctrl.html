Cerrando ventanas con wmctrl

Estoy utilizando un PC antiguo de tan solo 2Gb de ram. Aunque estoy utilizando el gestor de ventanas y i3wm, cuando abro Firefox y más de cuatro o cinco pestañas, comienza a ver una reescalada de RAM que llega a un lugar de no retorno y acaba bloqueando el PC.

Buscando información en internet, he encontrado el comando wmctrl que explica el amigo Joan del blog de Linux - Nuevas tecnologías y Linux en general

wmctrl proporciona una interfaz de línea de comandos para las tareas estándar de administración de ventanas, por lo que puede cambiar el tamaño y mover las ventanas, cambiar los escritorios, alternar los estados fijos, acumulados en una ventana y mucho más, todo desde un script

Instalación

Ubuntu, debian,...


sudo apt-get install wmctrl

Manjaro, Arch


sudo pacman -S wmctrl

Cerrando todas las ventanas

Si escribimos este comando en la terminal, nos cerrará todas las ventanas abiertas en el escritorio.


for xid in $(wmctrl -l | grep -e "-1" -v | awk '{print $1}'); do wmctrl -i -c $xid ; done

Explicación resumida del comando:

  • wmctrl -l Lista ventanas abiertas
  • wmctrl -i -c cerramos la totalidad de ventanas
  • awk '{print $1}' recorte primera columna

Ayuda

Encontrarás más ayuda con el comando:


wmctrl --help
