ntfy.sh otro sistema de notificaciones pub-sub simple basado en HTTP

ntfy  es un servicio de notificaciones HTTP pub-sub . Te permite enviar notificaciones a tu teléfono o escritorio a través de scripts desde cualquier PC o servidor, completamente sin registro ni costo . 100% código abierto pudiendo ejecutar el tuyo mismo, de momento solo para AMD64.

No deja de ser una alternativa a servicios como Gotify, Notificaciones via Telegram, etc...

Docker

El servidor lo instalaremos con:


docker run --rm -it -p 80:80 binwiederhier/ntfy

Recuerda que de momento solo hay arquitectura para AMD64.

Publicando

El sistema de publicación es PUT o POST. 

Publicar es tan sencillo como:


curl -d "Este es mi primer mensaje" ntfy.sh/raspberry

El modo es muy parecido a Mosquitto. El tema que he elegido es raspberry, pero recomiendo utilizar algo más complejo si utilizas el servidor público.

Conclusión

Es un proyecto muy nuevo pero muy interesante. Si estás interesado, te recomiendo visitar las ulrs que adjunto en el artículo y la web del proyecto https://ntfy.sh/.
