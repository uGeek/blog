Editando CSV usando Emacs

Tiempo estimado de lectura: 2 minutos. 253 palabras. emacs

Edito muchos archivos CSV (valores separados por comas) y aunque puedo hacerlo con un programa de hoja de cálculo, me resulta más rápido hacerlo con Emacs.

Emacs puede reconocer los archivos CSV gracias con modo csv. Podemos instalarlo mediante ELPA o descargando paquete desde el repositorio de GitHub y cargarlo en el arranque de Emacs, añadiendolo en el archivo de configuración (load-file "/path/to/csv-mode.el").

Con el modo csv, nos resalta las comas de otro color, pero para alinear o tabular las columnas y previsualizarlo como Excel, utilizaremos el atajo C-c C-a o introduciremos en Emacs M-x csv-align-fields

También podemos utilizar M-x toggle-truncate-lines, para deshabilitar la deformación de las líneas largas.

Línea Horizontal

Me gusta ver en que línea estoy trabajando para editar el archivo .csv. Para ello habilita M-x hl-line-mode.
