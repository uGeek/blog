Wireguard en Manjaro

Tiempo estimado de lectura: 1 minutos. 134 palabras. wireguard

Vamos a ver como instalar Wireguard en Manjaro

Instalación

sudo pacman -S wireguard-tools

Descargando el certificado

Descarga desde le servidor el certificado para el cliente y guárdalo en la ruta /etc/wireguard/.

Si renombramos el certificado con el nombre certificado.conf, iniciamos y detendríamos la conexión con el servidor con:

Iniciar

sudo wg-quick up certificado

Detener

sudo wg-quick up certificado

Por defecto

Por defecto el nombre que se suele utilizar para el cliente es wg0.conf

sudo wg-quick up wg0
