Actualiza el Firmware de tu Raspberry

Tiempo estimado de lectura: 1 minutos. 177 palabras. raspberry

Actualizar el firmware de tu Raspberry a la última, es muy sencillo.

Actualización

Para actualizar el firmware:

sudo rpi-update

A mi personalmente, me gusta poner todos estos comandos dentro de un script, así de un modo sencillo, lo pongo todo al día:

sudo apt-get update
sudo apt-get -y upgrade
sudo apt autoremove
sudo apt clean
sudo rpi-update
