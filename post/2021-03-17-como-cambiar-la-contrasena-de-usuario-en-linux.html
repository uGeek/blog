Cómo cambiar la contraseña de usuario en Linux

Tiempo estimado de lectura: 1 minutos. 206 palabras. bash

Para cambiar la contraseña de un usuario, podemos hacerlo desde la terminal. Vamos a ver tres ejemplos de como cambiar la contraseña: Cambiar la contraseña de tu usuario, cambiar la contraseña de otro usuario, siendo nosotros administradores y forzar a cambiar la contraseña a un usuario.

Cambiar la contraseña de usuario

passwd

Cambiar la contraseña de otro usuario

En este ejemplo, cambiaremos la contraseña del usuario luis.

sudo passwd luis

Obligar al usuario a cambiar la contraseña en el siguiente inicio de sesión

De forma predeterminada, las contraseñas están configuradas para que nunca caduquen. Para obligar a un usuario a cambiar su contraseña la próxima vez que inicie sesión, ejecutaremos el siguiente comando utilizando de ejemplo el usuario luis.

sudo passwd --expire luis
