Cambiar editor por defecto en Crontab

La primera vez que utilizas el comando crontab -e, te pedirá que elijas un editor por defecto y las siguientes veces que utilices este comando, siempre abrirá el editor que hayas elegido. Pero... ¿Como cambiar este editor?.

Para cambiarlo introduce el comando:


select-editor

También es posible añadir cambiar el editor creando del archivo .selected_editor en la raíz del $HOME del usuario.


nano ~/.selected-editor

En este ejemplo voy a utilizar micro como editor por defecto:


# Generated by /usr/bin/select-editor
SELECTED_EDITOR="/usr/bin/micro"

El comando te mostrará todos los editores de texto instalados en tu sistema. Si el editor que deseas utilizar no sale en la lista porque, por ejemplo utiliza otra paqueteria, editaremos el siguiente archivo, añadiremos al final:


nano ~/.bashrc

o si utilizas zsh:


nano ~/.zshrc

Para utilizar un editor que no tiene nada que ver con la terminal, por ejmplo typora, añade:


export VISUAL="typora"

Cambiar editor si nada de lo anterior funciona

Si quiere utilizar un editor en un momento especifico, en este ejemplo, con emacs:


sudo VISUAL=emacs crontab -e
