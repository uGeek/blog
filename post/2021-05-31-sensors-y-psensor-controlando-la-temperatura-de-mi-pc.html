Sensors y psensor. Controlando la temperatura de mi PC

Tiempo estimado de lectura: 2 minutos. 286 palabras. bash

Sensors y psensor, son aplicaciones que nos permiten conocer la temperatura de nuestra CPU, velocidad de los Ventiladores,…

./images-blog/bash.png

Vamos a ver como funciona

Instalación

En otras distribuciones el nombre del paquete puede cambiar, en ubuntu es lm-sensors.

sudo apt install lm-sensors

Funcionamiento

Ahora con un simple:

sensors

Nos mostrará la temperatura de nuestra CPU, temperatura crítica,en mi caso velocidad de los ventiladores…

coretemp-isa-0000
Adapter: ISA adapter
Core 0:       +29.0°C  (high = +76.0°C, crit = +100.0°C)
Core 1:       +29.0°C  (high = +76.0°C, crit = +100.0°C)
Core 2:       +29.0°C  (high = +76.0°C, crit = +100.0°C)
Core 3:       +29.0°C  (high = +76.0°C, crit = +100.0°C)

pc_smm-virtual-0
Adapter: Virtual device
Processor Fan:   1574 RPM
Motherboard Fan: 1166 RPM

Si no detecta la temperatura, podemos utilizar el siguiente comando para configurar sensors.

sudo sensors-detect

psensor

Si quieres conocer la temperatura de la CPU, GPU, discos duros, velocidad de ventiladores, memoria disponible… de modo gráfico, podemos utilizar psensor.\  

Para instalarlo utilizaremos:

sudo apt install psensor

Lanzar psensor

Para lanzar psensor, lo harimos así:

psensor

Ahora nos abrirá una ventana con toda la información antes citada.
