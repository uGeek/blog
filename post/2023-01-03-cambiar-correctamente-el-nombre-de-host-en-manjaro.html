Cambiar correctamente el nombre de host en Manjaro

Tiempo estimado de lectura: 0 minutos. 103 palabras. host

Para cambiar el nombre de host, de la máquina que estamos utilizando en ese momento, es tan sencillo como seguir los siguientes pasos:

Comprobar el nombre de host actual

Este comando nos mostrará el nombre de host entre otras informaciones como distro, kernel,…

hostnamectl

Cambiar el nombre del host

Para cambiar el nombre de host, sustituye NUEVO_NOMBRE por el nombre de host que desees:

``` sudo hostnamectl set-hostname NUEVO_NOMBRE
